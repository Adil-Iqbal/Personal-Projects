OOP Tic-Tac-Toe

This is an implementation of tic-tac-toe that is object oriented.
Each game of tic-tac-toe is resizable.
The game relies upon the p5js library.
The computer AI operates using the Minimax algorithm.
Other AI features include:
  1. Alpha-beta pruning.
  2. First-move random.
  3. Chance to blunder a move. (Between 0.1% - 15%)
